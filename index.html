Firebase Setup (5 min)

1
Go to console.firebase.google.com → Add project → name it anything (e.g. wc2026crew) → disable Google Analytics → Create.
2
In the left sidebar click Firestore Database → Create database → choose Start in test mode → pick any region → Enable.
3
Go to Project Settings (gear icon) → Your apps → click the </> Web icon → register app → copy the firebaseConfig object.
4
Open this HTML file in a text editor, find the section labelled 🔧 PASTE YOUR FIREBASE CONFIG HERE near the top, and replace each PASTE_YOUR_… value with the real values from step 3. Save the file.
5
Host the file on GitHub Pages (free) or drag it into Netlify Drop (free, instant) → share the URL with the crew.

Already configured? If you've already pasted in your Firebase config and saved the file, this screen will disappear automatically when you reload. You're seeing it because the placeholder values haven't been replaced yet.
